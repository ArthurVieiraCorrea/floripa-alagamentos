# Summary: Plan 09.1-02 — visualCrossingService.js

## What Was Built

`backend/src/services/visualCrossingService.js` — serviço que busca precipitação horária observada (mm) das últimas 24h do Visual Crossing Timeline API e persiste em `weather_observations`, atualizando `forecasts_meta.precip_48h_mm` com o total real observado.

## Key Files

### Created
- `backend/src/services/visualCrossingService.js` — 166 linhas; exporta `fetchAndCacheObservations()`

## Implementation Details

- **URL:** Visual Crossing Timeline API com janela de 24h em UTC-3 (`unitGroup=metric`, `include=hours`, `elements=datetime,precip,preciptype`)
- **Coordenadas:** Florianópolis (-27.5954, -48.5480)
- **Retry:** 3 tentativas com backoff exponencial (5s, 10s, 20s) via `AbortController`
- **Guard D-08:** Retorna silenciosamente se `VISUAL_CROSSING_API_KEY` ausente
- **No-throw D-10:** Erros de rede/parse logados, sem re-throw
- **Transação:** `BEGIN/COMMIT/ROLLBACK` garantindo atomicidade dos INSERTs
- **INSERT OR IGNORE:** Observações históricas são imutáveis (`fonte = 'visual-crossing'`)
- **D-11:** `UPDATE forecasts_meta SET precip_48h_mm = ? WHERE id = 1` executado APÓS o COMMIT
