**Modificação 2 — Cron horário expandido:**
```javascript
cron.schedule('0 * * * *', async () => {
  console.log('[scheduler] Executando fetch horário de previsão');
  await fetchAndCacheForecasts();
  await fetchVisualCrossing();   // NOVO
  await fetchRedemet();           // NOVO
```
